# إصلاحات مطلوبة - Success Stories

## المشاكل الحالية:
1. ✅ إضافة قصة جديدة - زبطت
2. ✅ صفحة التفاصيل - تم إصلاحها
3. ❌ إضافة Milestone - خطأ 500

## الإصلاحات المطلوبة:

### 1. ✅ صفحة التفاصيل - تم الإصلاح

**التعديلات يلي انعملت:**
- ✅ `injuryType` → `caseType`
- ✅ `treatmentDescription` → `storyDescription`
- ✅ `outcome` → حذف واستبدل ب `patientTestimonial`
- ✅ `gender` → حذف (مش موجود بالسكيما الجديد)
- ✅ `storyTitle` انضاف
- ✅ `patientTestimonial` انضاف
- ✅ تم إصلاح فورم التعديل
- ✅ تم إصلاح وضع العرض

### 2. مشكلة Milestone - جاري الإصلاح

**الخطأ:** 500 Internal Server Error

**السبب المحتمل:** الباكند ممكن يتوقع `storyId` كحقل بالـ FormData.

**التعديل يلي انعمل:**
- ✅ أضفنا `storyId` للـ FormData بالإضافة لإرساله بالـ URL

**البيانات يلي عم نبعتها هلأ:**
```javascript
{
  storyId: "uuid-من-الـurl",
  title: "عنوان المرحلة",
  description: "وصف المرحلة",
  date: "2025-01-15" (optional),
  imageUrl: File (optional),
  orderIndex: "0"
}
```

**لازم تجرب:**
جرب هلأ تضيف milestone وشوف لوق الـ console، وإذا ما زبطت، ابعتلي الخطأ يلي بيطلع.
